## 0.1.0 (2026-03-19)

### ✨ Features

* Initial release of Your Assistant v0.1.0 - A general-purpose real-time AI assistant
* Multi-provider AI support: OpenAI, Zai, Ollama, MLX, Anthropic, Gemini
* Multi-provider transcription support: Azure, Whisper, Web Speech, Deepgram
* Electron desktop app for macOS, Windows, and Linux
* Responsive web interface with mobile support
* Auto-mode with intelligent screenshot-based triggering
* Comprehensive documentation and testing setup
* Optimized CI/CD pipeline with Release Please automation
